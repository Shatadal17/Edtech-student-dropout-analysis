## Data Source

Dataset: Online Learning Student Dropout Dataset  
Source: Kaggle  
Link: https://www.kaggle.com/datasets/mohamedasak/online-learning-student-dropout-dataset 
Size: 5,000 rows | 15 columns  
Format: CSV

## How to Use

1. Download the dataset from the Kaggle link above
2. Save it as `student_dropout_dataset.csv` in this folder
3. Run the SQL scripts from the `sql/` folder to load into MySQL
